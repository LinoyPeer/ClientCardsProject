Project Title: My Business Cards
Creator Name: Linoy Pe'er

Overview:
"My Business Cards" is a React application designed to manage and display business cards. The app allows users to create, edit, delete, and like business cards. It features a user-friendly interface and responsive design, making it suitable for both desktop and mobile devices.

Features:
User Authentication: Users can log in and manage their cards based on their roles (admin, regular user).
CRUD Operations: Users can create new cards, edit existing ones, delete them, and like their favorites.
Responsive Design: The app adapts to different screen sizes using Material-UI components.
Search Functionality: Users can search for specific cards easily.
Tooltips: Informative tooltips enhance user interaction by providing context for actions.
Technologies Used
React: JavaScript library for building user interfaces.
Material-UI: React UI framework for building responsive applications.
React Router: For routing and navigation within the application.
Joi: For input validation.
Axios: For making API requests.
Local Storage: For persisting user data.

Getting Started:
Prerequisites
Node.js
npm (Node Package Manager)
